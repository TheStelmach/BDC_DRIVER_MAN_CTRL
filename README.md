# BDC_DRIVER_V2.0-MC
Simple brushed motor controller based on Timer 555

The board features two push buttons and two 20k variable resistors, for regulated bi-directional PWM control of a brushless DC motor.
MAX VOLTAGE: 5V
MAX CURRENT: 1A
